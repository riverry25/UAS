# Streamlit Deployment - Safety Helmet Detection

This folder contains a simple Streamlit-based deployment for the safety helmet compliance detection model.

## Purpose

The deployment allows users to upload an image and perform object detection using the trained YOLO model.

Detected classes:

- person
- helmet
- no_helmet

## Required Files

Before running the application, place the trained model file in this folder:

```text
Deployment/best.pt
```
